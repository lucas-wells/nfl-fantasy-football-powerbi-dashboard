# NFL Fantasy Football Analytics Dashboard

This project is an interactive Power BI dashboard analyzing fantasy football player production, route efficiency, first-read target share, and alignment-based usage roles.

## Project Overview

The goal of this dashboard is to identify how player usage and efficiency metrics relate to fantasy football production. The report uses a cleaned NFL player dataset with players who ran at least 100 routes.

## Tools Used

- Power BI
- Power Query
- DAX
- CSV data cleaning
- Data visualization

## Key Features

- KPI cards for total players analyzed, average PPR points per game, average yards per route run, and average first-read target share
- Position-based filtering for WR, RB, and TE
- Donut chart showing player distribution by position
- Bar charts comparing fantasy production by player, position, and alignment role
- Custom player alignment role classification using DAX
